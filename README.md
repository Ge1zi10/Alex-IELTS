# Alex IELTS Summer Learning Site

Static IELTS summer learning site for Alex.

## Files

- `index.html` - app shell
- `styles.css` - visual layout and responsive styles
- `course-data.js` - curriculum, daily lessons, prompts, exercises
- `vocab-data.js` - IELTS vocabulary source
- `app.js` - learning flow, scoring, local storage, dashboard

## Publish

This site can be published with GitHub Pages from the repository root.
